# Lessons Learned

## What I Learned

* How Microsoft Entra ID manages users and groups.
* Difference between administrative roles and security groups.
* Importance of naming standards in enterprise environments.
* How RBAC reduces the need for Global Administrator permissions.
* How organizations structure departments using groups.

## Challenges

Dynamic Groups were unavailable due to licensing restrictions.

## Resolution

Users were manually assigned to departmental groups to achieve the same business outcome.

## Future Improvements

* Implement Dynamic Groups.
* Configure Conditional Access.
* Enable MFA.
* Integrate Intune device management.
